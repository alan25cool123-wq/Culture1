О платформе

Наша миссия

Создать открытую платформу для обмена знаниями и литературой, где каждый может стать автором и читателем.

Как использовать

  1. Нажмите "Загрузить книгу"
  2. Заполните информацию о книге
  3. Загрузите текст книги в формате TXT или JSON
  4. Книга появится в списке для всех читателей
  5. Делитесь ссылкой на платформу с друзьями

GitHub Integration

Эта платформа может быть размещена на GitHub Pages. Данные хранятся локально в браузере пользователя с использованием LocalStorage и IndexedDB.

Для развертывания на GitHub Pages:

  1. Создайте репозиторий "username.github.io"
  2. Загрузьте эти файлы в репозиторий
  3. Откройте https://username.github.io

Конфиденциальность

Все данные хранятся в вашем браузере. Никакая информация не передается на серверы. Вы имеете полный контроль над своими книгами.
